# Pac-Man (Console Version)

A lightweight Pacman game implementation written in Java.  
This version runs entirely in the console and uses text-based rendering to display the game board.

---

## Gameplay
- Control Pac-Man using the keyboard:
  - **a** -> Move Left
  - **s** -> Move Down
  - **w** -> Move Up
  - **d** -> Move Right
- Eat pellets to increase your score.
- Avoid ghosts.
- Clear the board to win.

---


## Running the Game
Make sure you have Java installed (JDK 8 or higher recommended).

Compile the game:

```bash
javac game.java
